Importar Planilha

Selecione uma planilha Excel (.xlsx ou .xls) para importar produtos em massa. Use o modelo padrão para garantir compatibilidade.

Baixe o modelo de planilha antes de importar
  • Campo ID — Refere-se ao código do produto no Sistema Raízes do Nordeste. Se o ID informado já existir no sistema, o produto será atualizado. Caso contrário, um novo produto será criado. Para adicionar novos produtos, não é necessário preencher o campo ID.
  • Campo idReferência — Corresponde ao ID do produto no sistema terceiro (ERP, PDV, etc). Use este campo para manter o vínculo entre os sistemas.
